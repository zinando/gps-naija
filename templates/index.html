What is API?

API is a way that allows two or more computer programmes to interract with each other. This interraction usually leads to data flow from one of the computer programmes (the API provider) to the other/others (the clients) or in both directions. The API host computer programme usually have resources/information needed by other computer (clients) programmes. So, it provides an interface of communication to allow these clients to get these resources from it. The word API is an acronymn that stands for Aplication Programming Interface.

In the context of our API, gps-naija has a datastore of the names of all the States in Nigeria and their capital, their Local Government Areas (LGA), major landmarks in each LGA, and Streets. If your application is going to need this information for it to serve its purpose, then this API will be usefull to you. You will not need to hard-code the names of these places in your application, just follow the instructions below to get the data that you need.

Endpoints To Get Data For All The States In Nigeria

curl https: //gps-naija.onrender.com/states

- H"Content-Type: application/json"

- X GET

For example, in a python code, i would write:

#import requests module

import requests

url = 'https://gps-naija.onrender.com/states'

headers = {'Conten_Type': 'application/json'}

response = requests.get(url, headers = headers)

Endpoints To Get Data For One State In Nigeria

curl https: //gps-naija.onrender.com/states/<state name>

- H"Content-Type: application/json"

- X GET

For example, to get data for Lagos State, in a python code, i would write:

#import requests module

import requests

url = 'https://gps-naija.onrender.com/states/lagos'

headers = {'Conten_Type': 'application/json'}

response = requests.get(url, headers = headers)
